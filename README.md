# openmotion-mechanical
Repository for mechanical engineering files for Open-Motion

# Overview
Mechanical files for Open-Motion will be uploaded when major releases occur.

Open-Motion is still in development and files here represent prototypes for the two major components: the console and the patient applied part. The console which contains the laser, power supply, and control circuitry. The patient applied part contains the array of sensors and the laser output.

Please see the other repositories in this organization for the Electrical and Software files.
